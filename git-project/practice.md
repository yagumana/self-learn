# git-tutorial

1. branch名の変更
    - コマンドパレットを開く(Shift+ctrl+p)
    - rename branchを選択
    - branch名を入力(ex. test-branch)

2. ファイルの修正
    - branch名をmainに変更し、行を追加すると変更行が緑になるのがわかる
    - 行を削除すると、赤くなる
    - source control(Ctrl+Shift+G)から、ファイルを指定すると、変更箇所が見れる

3. mergeのやり方
    - mainブランチに戻る
    - source controlのドットから、mergeを選択
